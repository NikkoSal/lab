# Метод наискорейшего градиентного спуска

Данный репозиторий содержит реализацию метода наискорейшего градиентного спуска для решения задачи безусловной минимизации дифференцируемых функций.

Проект выполнен в учебных целях и может использоваться как пример численной реализации классического метода оптимизации.


## Постановка задачи

Рассматривается задача минимизации функции  
\( f(x) \), где \( x \in \mathbb{R}^n \).

Предполагается, что функция:
- является дифференцируемой;
- имеет непрерывный градиент.

Целью является нахождение точки минимума функции.


## Идея метода

Градиент функции  
\( \nabla f(x) \)  
указывает направление наискорейшего возрастания функции.

Следовательно, движение в направлении  
\( -\nabla f(x) \)  
обеспечивает наискорейшее убывание значения функции.

Метод наискорейшего градиентного спуска отличается от обычного градиентного спуска тем, что шаг на каждой итерации выбирается оптимальным образом.


## Итерационная схема

Алгоритм строит последовательность точек по формуле:
\[
x_{k+1} = x_k - \alpha_k \nabla f(x_k)
\]

где шаг \( \alpha_k \) находится как решение одномерной задачи минимизации:
\[
\alpha_k = \arg\min_{\alpha > 0} f(x_k - \alpha \nabla f(x_k))
\]

Для поиска оптимального шага используется метод золотого сечения.


## Критерий остановки

Итерационный процесс завершается при выполнении условия:
\[
\|\nabla f(x_k)\| < \varepsilon
\]

что соответствует необходимому условию минимума функции.

## Реализация

Метод реализован на языке **Python** с использованием библиотеки **NumPy**.

Код включает:
- реализацию метода наискорейшего градиентного спуска;
- одномерный поиск шага методом золотого сечения;
- набор тестовых функций для проверки корректности работы алгоритма.

В рамках проекта проведены тесты для следующих функций:
