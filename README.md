# Метод наискорейшего градиентного спуска

В данном репозитории представлена реализация метода наискорейшего градиентного спуска для решения задачи безусловной минимизации дифференцируемых функций.

Проект выполнен в учебных целях и предназначен для демонстрации принципов работы одного из базовых методов численной оптимизации.

---

## Постановка задачи

Рассматривается задача поиска точки минимума функции нескольких переменных. Предполагается, что функция является дифференцируемой и имеет непрерывный градиент.

Цель алгоритма — найти такую точку, в окрестности которой значение функции перестаёт уменьшаться.

---

## Идея метода

Метод основан на использовании градиента функции. Градиент указывает направление наискорейшего возрастания значения функции, поэтому движение в противоположном направлении приводит к её убыванию.

На каждой итерации алгоритм перемещается из текущей точки в направлении антиградиента, выбирая такой шаг, при котором значение функции уменьшается максимально возможным образом.

---

## Принцип работы алгоритма

Работа алгоритма состоит из следующих шагов:

1. Задаётся начальная точка.
2. Вычисляется градиент функции в текущей точке.
3. Определяется оптимальный шаг движения вдоль направления антиградиента.
4. Выполняется переход в новую точку.
5. Процесс повторяется до тех пор, пока градиент не станет достаточно малым.

В качестве критерия остановки используется малая норма градиента, что соответствует необходимому условию минимума функции.

---

## Поиск шага

Оптимальный шаг на каждой итерации находится с помощью одномерного поиска минимума. В данной реализации для этого используется метод золотого сечения, который обеспечивает устойчивое и надёжное уменьшение значения функции.

---

## Реализация

Алгоритм реализован на языке Python с использованием библиотеки NumPy. Реализация не использует готовые оптимизационные библиотеки и полностью соответствует классическому описанию метода наискорейшего градиентного спуска.

---

## Численные эксперименты

Для проверки корректности работы алгоритма выполнены численные эксперименты на нескольких тестовых функциях, включая простые квадратичные функции и более сложные нелинейные примеры.

Полученные результаты демонстрируют быструю сходимость метода для простых задач и замедление сходимости для плохо обусловленных функций, что соответствует теоретическим свойствам метода.
